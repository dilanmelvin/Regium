Install

npm install @regium/core @regium/country-in @regium/country-us
# or
pnpm add @regium/core @regium/country-in @regium/country-us

Quick start

import { createRegium } from "@regium/core";
import india from "@regium/country-in";
import us from "@regium/country-us";

const regium = createRegium({ plugins: [india, us] });

// Country lookup
regium.getCountryConfig("IN");
//   → { name: "India", iso2: "IN", currency: { code: "INR" }, ... }

// Validate a tax ID
regium.validate({ country: "IN", field: "PAN", value: "ABCPL1234C" });
//   → { ok: true, normalized: "ABCPL1234C" }

// Generate localized employee fields
regium.getEmployeeFields("US");

Metadata model

Every metadata document in Regium is wrapped in an envelope that carries the version, effective date range, deprecation status, and source citations. This lets you reproduce historical payroll runs and audit every legal change. 

interface MetadataEnvelope<T> {
  id: string;
  domain: "country" | "payroll" | "tax" | ...;
  jurisdiction: string;
  version: string;
  effectiveFrom: ISODate;
  effectiveTo?: ISODate;
  deprecated?: boolean;
  source: SourceCitation[];
  data: T;
}

Versioning

Three independent axes:

  • Package semver — API contract.
  • Metadata snapshot — calendar tag YYYY.MM.
  • Effective dates — per-rule legal validity.

Plugins

Country packs and custom validators register through a host:

const regium = createRegium({
  plugins: [
    india,           // bundled country pack
    {                // custom plugin
      name: "@acme/regium-private",
      version: "1.0.0",
      setup(host) {
        host.registerValidator(myCustomValidator);
      },
    },
  ],
});

@regium/validators

Composable, deterministic, framework-agnostic. Every validator is pure.

import { iban, swift } from "@regium/validators";

iban.validate("DE89 3704 0044 0532 0130 00");
// { ok: true, errors: [], normalized: "DE89370400440532013000" }

@regium/forms

Build a form schema from compliance fields, then render with any framework.

import { buildFormFromFields } from "@regium/forms";

const schema = buildFormFromFields({
  jurisdiction: "DE",
  audience: "employee",
  fields: regium.getEmployeeFields("DE"),
  locale: "de-DE",
});

@regium/payroll

import { computePayroll } from "@regium/payroll";

const result = computePayroll({
  annualGross: 1200000,
  rules: regium.getPayrollRules("IN"),
  taxRules: regium.getTaxRules("IN"),
});

result.netMonthly;    // → number
result.deductions;    // → PayrollLine[]
result.totalEmployerCost;
